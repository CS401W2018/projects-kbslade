Internet

The internet is a group or network of computers and electronic devices that share information and communicate with one another.

Web

A system that allows us to share information and content over the internet. It's the documents and links that are shared over the internet, or network. Another name is the World Wide Web (or www).

How are the Internet and Web different?

The Internet is the network that connects the computers and devices whereas the web is the information, links, and databases.

ASCII and Unicode

Unicode is universal characters used on a computer that translates binary to language we humans can understand. ASCII is 7 or 8-bit binary numbers that represent English, so we can communicate with the computer.

What is UTF-8?

UTF-8 is an encoding system used in Unicode. It translates numbers to binary

URL

It's a web address. It stands for Uniform Resource Locator.

Domain Name

A domain name is a website's name.

HTTP

Stands for Hypertext Transfer Protocol. It is a request and return system.

HTTPS

The same as HTTP but with added security, which is what the "S" stands for.

IP Address

IP stand for "Internet Protocol". It is able to identify the device connected to and using the internet.

DNS

DNS stands for domain name system. It connects web browsers and websites.

W3C Recommendations

It's a guideline for the web. It standardizes the web.

XML, XHTML, HTML, HTML5

XML is a markup language that provides rules. XHTML extends versions of XML. An example of this would be ".xhtml". HTML is the standard markup language for the web. HTML5 is the fifth version (and final version thus far) for HTML.
