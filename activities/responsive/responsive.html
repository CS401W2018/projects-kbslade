Why is the viewport meta tag important?

The viewport meta tag is in the html text above the body (in the head category). It tells the page that no matter what the type of device that the person is using (i.e. mobile, desktop, etc.), the page will adjust to the width of the device.

When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

The one you give last is the one that it'll do. You could also add the word "important" after the direction and it'll force it to do it, - so in the example, "flex-direction: column important", it'll make the column even though the row is different.

How do you change the CSS for a web page displayed on a phone?

You add a media tag ("@media only screen and (max-width:800px), then set the parameters you'd like when it's on a mobile device.")

Notes: The content for the mobile phone would be first. Check the mobile view in the browser using the "inspector" tools. You can select different device views.
