Why don't we use the border property?

We don't use border property in html because it won't allow us to "style" it in CSS. It's something they used in the past, but it's very inflexible, especially to mobile devices.

What is the easiest way to color every other row a different color?

you can specify having every other row (odd or evens) in the CSS sheet. i.e. .schedule thead, .schedule tfoot { background-color: aquamarine } .schedule tbody tr:nth-child(even) { background-color: blueviolet; }

When should we use a table?

To display data, like for stats, your schedule or other info.

Extra notes:

  • border-spacing and border-collapse only apply to tables.
  • .schedule td, .schedule th you can add another part, like "th" to the line with the "td" to get them styled the same. You just have to add it with a comma. Also, you'd want to specify which one if you have more than one table on your site and want them to be different or specific to that one table.{ border: thin solid black; padding: 10px;}
  • To label your table, you can use a caption tag. However, to move the caption, like to the bottom, you can only do that in CSS with "Caption-side:"
