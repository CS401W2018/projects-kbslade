Extra Notes and Engagement Questions:

  • the "?"in the address is the query string
  • "radio" means 1, meaning it'll select only one
  • checkbox is one input (like y/n)

The form tag has an "action" attribute. What does it do?

The action attribute allows you to input information and do something.

The form tag has a "method" attribute. What does it do?

The method attribute in the form action line, can be method="get"; can also do method="post" and it won't show the info on the browser line. But is still not secure. You need the "s" on https. It makes it encrypted.

What is the difference between a "POST" and a "GET"?

  • We use a "get" request when you want to read (or get) something. Looking up something on Google is a good example. (The words you used to look it up are in the browser line.)
  • We use a "post" request when you are changing things on the server, like creating an account, when you submit your assignment, when you're adding info to the server.
