Why is the viewport meta tag important?

The viewport meta tag is in the html text above the body (in the head category). It tells the page that no matter what the type of device that the person is using (i.e. mobile, desktop, etc.), the page will adjust to the width of the device.

How can you adjust the CSS for printing?

You create a separate css stylesheet to print "print.css" tagged with media="print". You copy your main .css page and change the few items to make it more print friendly.

When would you want to adjust the CSS for printing?

You may want to change the font to a serif font, if your page is in a sans-serif font, to print easier. You can take out things, like the navigation bar (set it to "display: none;), since you don't need links on a printed page. That way, when the person wants to print the page, it'll be adjusted specifically for printing.

You'd add the print.css and adjust it for printing when the site is something that people would need to print, like tickets or cards.
