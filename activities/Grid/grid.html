What is the main difference between CSS Grid and CSS Flex?

A grid is a lot like a table with rows and cells. The grid sets up the rows and columns.

In Grid: Parent is not flexible. Dictates where the "kid" (items) go. In Flex: The kid gets to determine their own size. Flex is easier.

Other notes:

When adding a grid, you need to set the display to grid. Then, on the next line, in your CSS: "grid-template-columns: 1fr 1fr 1fr;", the number of 1fr you add is the number of columns you want. In this example there are 3 "1fr", so there are 3 columns. The 'fr' stands for fraction, so it divides the columns evenly. If you want one column wider or less wide than another, then you can put in your own px length, like "150px 300px" - it'll still have 3 columns, but the first will be 150px, the second 300px, and the third will be as wide as the page/parent.
