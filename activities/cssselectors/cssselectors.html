Ways to select a tag

Read on w3 schools for more info.

    Selector Rules:
  1. Tag name
  2. Descendent (or children) - main p (only targets paragraphs in main)
  3. ID: id= _____ (name) and you give it a name so you can identify the tag. Then, in CSS, you use the #(name- spelled exactly the same and in the same case, no spaces). ID can only be used ONCE! Ususally use a noun name.
  4. Class -.: class is similar to ID, but it can be used more than once or reused. A lot of times you use an adjective to name it.

A Span tag: doesn't do anything on it's own, nor does it split up the line. It separates or "highlights" text so you can add a feature to that select text.

A DIV tag: Causes line breaks! Everything in the div tag spans the page. It creates a box on the page, or a block. Div tags came before the header, nav, main, and footer tags. Those custom tags act as div tags.
